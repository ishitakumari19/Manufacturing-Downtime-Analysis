  ## Key Business Insights

1. Machine M12 contributes the highest downtime across the factory.

2. Mechanical failures account for the largest share of machine breakdowns.

3. Night shifts experience lower production output and higher downtime.

4. Increased vibration levels are strongly associated with machine failures.

5. Corrective maintenance results in significantly longer downtime compared to preventive maintenance.

## Recommendations

- Increase preventive maintenance frequency for critical machines.
- Install vibration monitoring alerts for early failure detection.
- Improve staffing and supervision during night shifts.
- Prioritize replacement of high-failure components.
- Develop predictive maintenance schedules using sensor data.
